About BIASDB

History

The project started in 2019 with a goal to provide quick access to some useful information about uncommon and common bacteria identified by automated systems for clinical microbiologists to use when they need to evaluate an identification result. The first version of the database contains mostly data collected from online resources such as PubMed and Wikipedia via an automated text mining pipeline. Although the extracted information is quite useful, we are currently expanding our source of data to other online materials and also clinical microbiology laboratories in Thailand to add more data to the database.
